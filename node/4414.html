dastin / 12.12.2014, 11:17

ну тогда вот вам
примерчик

Sub pagesToBitmap()
    Dim p As Page, sr As ShapeRange, s As Shape
   
    For Each p In ActiveDocument.Pages
        p.Activate
        Set sr = ActivePage.Shapes.All
        If sr.count > 0 Then
            Set s = sr.Group
            s.ConvertToBitmapEx cdrRGBColorImage, False, False, 150, cdrNormalAntiAliasing, True, False, 95
        End If
    Next p
End Sub

отредактируйте под себя и вперёд
