Обрезка изображения при импорте PS файла

MBK / 20.09.2012, 01:40/00:41
Форум: 
CorelDRAW

В ряде случаев, при импорте Postscript interpreted файла в Corel наблюдается неприятный глюк - импортируемое изображение обрезается по размеру примерно 216х276 (letter) начиная с левого нижнего угла. Такое наблюдается, когда импортируемый PS файл был напечатан на нестандартный Custom формат бумаги. Это происходит от того, что в некоторых PPD кастомный формат реализован не через setpagedevice, а через setpageparams, а данная функция, судя по всему, кореловским интерпретатором отрабатывается некорректно. Данный баг существует, как минимум, в ряде билдов версий X3-X5 без подключенного GhostScript.
Чтобы импортируемое изображение не обрезалось, необходимо задать для него заведомо максимальные размеры полей обрезки. Для этого необходимо найти и отредактировать файл startup.ps Обычно, этот файл находится в каталоге Data или Programs/Data установленного пакета. Открываем этот файл в текстовом редакторе и ищем строку 

PAPERSIZE cvx exec

Эту строку необходимо заменить на

<<	
	/PageSize [100000 100000]
>> setpagedevice

После этого сохраняем поправленный файл, перезагружать корел при этом не обязательно.
