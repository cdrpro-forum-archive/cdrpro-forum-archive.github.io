Программа использует только установленные шрифты в системе. Это вовсе не значит что они обязательно должны располагаться в папке шрифтов системы. Свои шрифты можно хранить в любой другой папке, а устанавливать их нужно с помощью специальных программ, например FontExpert.

П.С. Шлак типа AdobeTypeManager лучше не использовать, а забыть навсегда как страшный сон.

Еще есть Bitstream Font Navigator, который идет в поставке с CorelDRAW. Тоже программа ничего, можно пользоваться. CorelDRAW, кстати, может использовать базу данных Font Navigator'а, автоматически инсталлируя необходимые для конкретного документа шрифты. Прямо при открытии этого документа.
