Наиболее оптимальным в данном случае является формат PDF. B CorelDRAW воспользуйтесь командой Publish to PDF (Опубликовать в PDF) c выбором стандартного набора настроек For PrePress. B выпадающем списке поля Advanced придется выбрать пункт Output All Objects as: Native (Вывести все объекты как: Родные). Затем сохраненный файл можно будет открыть непосредственно из Photoshop, воспользовавшись командами Import > PDF Image (Импорт > РDF-изображение) или Automate > Multi-Page PDF to PSD (Автоматизация > Многостраничный PDF в PSD). Также этот же файл можно будет открыть через Adobe Illustrator, откуда его сразу же экспортировать в форматы TIFF или PSD. Возможно, что после таких операций вам будет необходимо несколько подправить прозрачность в сохраненных файлах.
