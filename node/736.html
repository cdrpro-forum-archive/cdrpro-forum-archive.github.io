Системные требования:

* Microsoft® Windows® 7, Windows Vista®, Windows® XP с последним пакетом обновления (32-разрядная или 64-разрядная версия)
* Intel® Pentium® 4, AMD Athlon 64 или AMD Opteron
* 512 Мб оперативной памяти (рекомендуется 1 Гб)
* 750 Мб свободного места на жестком диске (1 Гб для обычной установки без контента)
* Мышь или планшет
* Разрешение экрана 1024 x 768 (768 x 1024 на планшетном ПК)
* DVD-дисковод
* Microsoft® Internet Explorer® 7 или более поздней версии

dastin / 20.06.2010, 13:41

Особенности всегда одни и те же. Памяти побольше, проц покруче, Х5 теперь мультикоре, осталось разобраться с скретч-дисками.
Стандартная ситуация
- винда со своими свопфайлом (pagefile.sys)
расположение и размер которого настраивается из пункта
Мой компьютер-Свойства-Дополнительные параметры-Быстродействие-Дополнительно-Виртуальная память ( по умолчанию на системном диске)

- каталогом темр :k:\Documents and Settings\Users\Local\Temp\
расположение которого настраивается из пункта
- Мой компьютер-Свойства-Дополнительные параметры- переменные среды
( по умолчанию на системном диске)

- скретч-диск фотошопа - расположение которого настраивается из соответствующего пункта преференсов шопа
( по умолчанию на системном диске)

- скретч-диск Корела - Tools-Option-Memory (Инструменты-Параметры-Память)
( по умолчанию на системном диске)

так вот - желательно, чтобы все эти дисковые пространства имели достаточный запас и были разведены по разным логическим ( а лучше физическим) дискам, чтобы при одновременной работе шоп, корел и винда не обращались на один и тот же диск для записи своих временных файлов

1. Параметры > рабочее пространство > память

Есть пункт "Доступный объем ОЗУ" — я ставлю половину или чуть больше, но стараюсь ПОЧТИ ВСЕГДА работать одним Корелом, не запуская больше никаких других программ.
(кстати там выставляется и адрес файла подкачки Корела, про который Вам так хорошо рассказали)

2. А у вас есть тени и другие эффекты? По умолчанию Корел все это делает в 300 dpi.
Если у Вас наружка, можно смело уменьшать:
Параметры > рабочее пространство > общие
пункт "Разрешения отбражение".
