Из стандартных средств можно только облегчить задачу: выделить один объект, смасштабировать, выделить другой объект, нажать Ctrl+R. Масштабирование повторится. Конечно, придется выделять все объекты по одному, но процесс ускорится в разы. (Правда в 13 кореле нет возможности попробовать, но должно работать)

dastin / 06.05.2014, 23:54

Transform Each — макрос для массовой трансформации объектов в CorelDRAW

Состоит из трех вкладок:

Scale
Rotate
Skew

В первой вкладке Scale имеются два поля ввода — ширина и высота. Вводимые значения должны быть в миллиметрах. Есть возможность вводить проценты — флажок Use percent %. Для пропорционального масштабирования одно из полей оставляйте пустым.
