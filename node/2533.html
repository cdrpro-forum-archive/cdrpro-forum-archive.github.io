Baryga88 / 08.09.2011, 16:09

Можно такое сделать двумя способами. Перетеканием вдоль пути. Задать только нужно необходимое количество объектов (рисок). Или, как было уже сказано дублированием с одновременным поворотом на нужный угол. В обеих случаях, нужно центр вращения поместить в центр "мнимой окружности". Дублировать можно с помощью мыши, задав в параметрах нужный угол и вращать при нажатой клавише Ctrl и потом не отпуская левую кнопку, щелчок правой (как при обычном копировании мышью). Сделали одну такую копию риски, потом Ctrl + R (повтор команды). Корел сам потом будет делать копию риски с одновременным поворотом на нужный угол.
