Проблемы с цветом в corelDRAW. Как быть?

Для корректного отображения цвета, нужно настроить Color Management System. Подробности можно прочитать в соответствующем разделе нашей библиотеки: Управление цветом в CorelDRAW.

Sancho / 16.07.2011, 17:35/17:36

Файл не открывается. Как его восстановить?

  • Попробовать импортировать сбойный файл в новый пустой документ CorelDRAW.
  • Попробовать отрыть файл на других компьютерах/в других версиях программы (более новых).

Если ничего не помогает, можно воспользоваться программой CDRFIND.

Файл открывается долго! Копирование и вставка тормозит! Файл весит много хотя в нём простой вектор. В чём дело?

Самые распространённые проблемы — цветовые стили и стили просмотра. ссылки по теме:
Решаем проблему цветовых стилей (Color Styles)
Удаление цветовых стилей в CorelDRAW
Удалить стили просмотра можно с помощью функции sToolsDocument.DeleteViewStyles из макроса cdrpro.ru/cdrtools

Из за чего ещё файл может весить много?

Возможно следующее:

  • В документе есть символы в библиотеке символов
  • Документ содержит скрытые слои с объектами или они просто лежат далеко за пределами листа.

Почему объект оставляет много копий после себя при перетаскивании?
Проще говоря — залип пробел на клавиатуре. Дело в том что в "нормальном состоянии" железа, перетаскивая объект и нажимая пробел, вы дублируете объект — одна из функций CorelDRAW.
