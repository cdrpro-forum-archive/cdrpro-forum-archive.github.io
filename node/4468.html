Создаём перетекание линий (с заданными заранее параметрами)

Разъединяем группу с перетеканием Ctrl+K,
Отменяем группировку полностью,
Объединяем Ctrl+L,
Копируем в буфер обмена Ctrl+С,
Набираем и размещаем необходимый текст

Удаляем основной объект (наш текст) из фонового

Вставляем из буфера обмена Ctrl+V,
Уменьшаем толщину линий приблизительно в 2 раза (для наглядности задал линиям другой цвет)
Группируем Ctrl+G

Пересекаем с объектом (прямоугольник, окружность и т.д), можно подровнять с помощью контейнера

Удаляем лишнее. При необходимости можно преобразовать линии в объект

Спасибо за внимание.
