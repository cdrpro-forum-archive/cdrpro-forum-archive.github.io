Проблема с сервис паками CorelDRAW X6

Stamp2004 / 04.04.2013, 23:56
Форум: 
CorelDRAW

Куплен CorelDRAW X6 (коробочная версия), система Windows 7 32-bit. Установлен успешно, зарегистрировался на corel.com. Позже были установлены сервис пак 1, сервис пак 2, и hotfix для сервис пак 2. Обрадовался выходу сервис пак 3, но... Вот тут и пошли проблемы. Сервис пак 3 был успешно скачан и установлен. И вдруг пропала реакция на "Инвертирование". Выделяю объекты, задаю инвертировать - ничего! (Функция для меня ОЧЕНЬ важна!). Решил откатиться назад - не вышло. Снес Корел, почистил реестр, переустановил Корел. Но теперь не устанавливается сервис пак 1 - при установке пишет "сайт corel.com временно недоступен", хотя на сам сайт открывается и под своей учетной записью зайти могу в браузере. В чем проблема, может мой косяк?

Stamp2004 / 15.07.2013, 15:21

С апреля месяца веду переписку с Corel по поводу временно недоступного сайта. Общий сбор информации (где купил, какой серийник, свойства системы...) и никакого вразумительного ответа. Жаль...

Stamp2004 / 19.09.2013, 11:38

Наконец-то получил ответ от службы поддержки: "Здравствуйте, попробуйте задать вашему компьютеру имя состоящее из латинских букв, и повторите попытку установки.
Информацию о смене имени машины вы можете найти в документации вашей операционной системы.
Если по каким-то причинам нежелательно менять имя машины, вы можете дождаться обновления,
которое решает данную проблему. предположительная дата выхода обновления - следующая рабочая неделя".
Заработало!!!
