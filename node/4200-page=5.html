XmasterX / 22.09.2014, 23:51

пошагово (проверять на документе с кол-вом листов >1):
- группа из объектов за пределами листа, на десктопе (или один объект)
- нажиммаем «доп +» для создания дубликата
- дубликат автоматом выделяется, к нему шифтом добавляем (выделяем) объект на листе
- выравниваем дубликат относительно второго объекта, по центрам, кнопками С и Е
- дубликат ложится на лист, но в диспетчере объектов он остается на десктопе - изза этого его нельзя передвинуть за объекты лежащие на листе

win7 x64, corel со вторым SP

сравнение с X5, там нормально - как только объект появляется над листом, он перемещается в laуer1 этого листа
