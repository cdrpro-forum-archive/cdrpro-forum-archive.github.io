эти параметры есть в реестре
HKEY_CURRENT_USER\Software\Corel\CorelDRAW\17.0\Draw\Application Preferences\Artistic Media Tool\ObjectSpraySpacing

а также в секции Artistic Media Tool файла
c:\Users\***\AppData\Roaming\Corel\CorelDRAW Graphics Suite X7\Draw\Workspace\settings.ini

вот только их изменение ни к чему не приводит
после перезапуска программы - снова здорово - 25,4

возможно что, эта цифра забита в библиотеке
*:\Program Files\Corel\CorelDRAW Graphics Suite X7\Programs\ArtisticMediaTool.dll
