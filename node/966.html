dastin / 14.02.2011, 20:08

Window - Color Palettes - отключить Default RGB - включить Default CMYK - по правой клавишей на самой палитре Default CMYK, вверху, где квадратик с треугольничком - вызвать контекстное меню и в нём поставить Set as Default - после этого у вас в окнах градиентов (и в других местах) будет CMYK
Удачи.

dastin / 23.06.2011, 21:30

Если вы отключили палитру RGB
Window - Color Palettes - Default RGB - снять галку
- никаких палитр у вас показываться не будет - так?

Если после этого включите палитру CMYK
Window - Color Palettes - Default CMYK - поставить галку
- то однозначно вы будете видеть только её - так?
далее нажмите любой клавишей вот эту кнопочку
( см. рисунок)
найдите и нажмите там Set as Default
