Нужно скопировать полосы для написания текста в многостраничном документе, как это сделать автоматически?

dastin / 23.11.2013, 13:50/14:03

у вас N страниц ... готовите в Экселе таблицу-"рыбу" с N текстовыми полями, сохраняете как csv.
Далее в CorelDRAW создаете новый документ с необходимыми параметрами - размер, ориентация ...
на его основе делаете слияние при печати (Print Merge) - цепляете вашу табличку - после вставки поля преобразуете артистик текст в параграф, настраиваете фрейм по размеру, положению, выравниванию, шрифту, отступам, стилю .... и жмете на панели Print Merge - Merge to New document - получаете N страниц с фреймами параграф текста.

если текст на всех страницах должен быть одинаковым - проще будет поместить его на мастер-страницу
