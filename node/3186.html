У меня есть не праздный вопрос.
Есть большой файл с множеством мелких объектов на большой площади. Мне нужно скопировать объект в одном конце файла и вставить в другом. По дефолту объект скопированный по Ctrl+C по Ctrl+V вставляется в том же месте поверх копируемого объекта. Порядок действий Скопировать-Вставить-Перетащить очевиден и не удобен в некоторых случаях.

Например в Иллюстраторе скопированный по Ctrl+C объект по Ctrl+V вставляется в том месте которое в данный момент отображается на экране в центр экрана, а по Ctrl+B объект вставляется в то место где был скопирован под копируемый объект.

Есть ли что-то подобное в Corel Draw?

Sancho / 14.01.2013, 18:05
