Here is what I used to fix my old macros, so they will work in X6-64 bit version. Open the macro editor and run the macro, then find the lines that are usually in red (declare function error has occured at this point)

Change this:

Private Declare Function RegOpenKeyEx Lib "advapi32".....(or whatever your private declare function is)

To this:

Private Declare PtrSafe Function RegOpenEx Lib "advapi32".......

Inserting the PtrSafe allows the 64 bit systen to recognize the program command. I don't know if this will fix every macro with a private declare function, but give it a try and see if it works.
