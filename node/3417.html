Автоматизация размещения документов на одном листе

alexeyvanzhula / 03.08.2013, 21:55/01:48
Форум: 
Макросы и скрипты

Есть 7 больших ( около 1м х 2м) рисунков\документов одинакового содержания, отличающихся лишь чють-чуть размером по горизонтали.
Как сочинить макрос или еще что нибудь для того чтобы все они автоматом размещались на листе А4 (в ряд.).
То есть из действий нужно примерно:
1. Уменьшить размер документа.
2. Обвести рамкой границы документа - то есть обрамить содержимое ректанглом
3. Сгрупировать все.
4. Закинуть на лист А4 - по порядку начиная с найменьшэй ширины.
5. Закрыть документ.

Нужна автоматизация, так как таких папок ( с 7 рисунками ) очень большая куча.
