ак етим пользоватся

Sokolovich / 26.01.2013, 19:54/19:55

В открывающейся по клику на иконке extrusion lighting палитре есть окошко с превью условного трехмерного объекта. Слева расположены 3 пиктограмки лампочек. По умолчанию ни одна из них не активна. Если щелкнуть по одной из них, в окошке отобразится ее положение относительно объекта. Можно активировать все три. Перемещать их можно в окошке превью по узловым точкам сетки "кубика". Слайдер внизу задает яркость выбранной в данной момент лампочки. Галка Full Color Range, насколько я понимаю, позволяет использовать всю гамму полутонов от белого (максимально яркий свет) до черного (отсутствие подсветки)
