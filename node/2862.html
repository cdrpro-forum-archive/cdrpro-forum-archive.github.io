Однажды у меня возникла необходимость создать несложный сайт-портфолио. Специальными программами для этого я не владею.
Покопавшись в Кореле, я обнаружил замечательные возможности для создания работающего сайта прямо в CorelDRAW. На выходе получаются готовые html-файлы и папки с изображениями, т.е. можно просто опубликовывать в сети и все.

Достигается это через меню window => toolbars => internet

Любому объекту CorelDRAW можно назначить ссылку в поле URL. Это могут быть html файлы, изображения или ссылки на другие сайты.

Потом через меню file => publish to the web => html генерерируются готовый файлы сайта.

Очень удобно и практично!

В Corel дизайн сайта можно делать абсолютно любой. После экспорта, Корел всю графику переводит в растр (можно выставлять режимы, качество, формат и т.п.), а все активные объекты, ссылки и т.п. прописывает в html файле, в виде кода. Насчет индексации поисковиками, видимо ты прав, но я не особо разбираюсь в этом.
Точно есть возможность оставить текст в виде текста. На тулбаре Internet, кнопка Make text Web compatible.
Кстати в этом же тулбаре Internet есть функция Create rollover - анимационные кнопочки делает. Технически они типа Кореловского контейнера, в котором несколько видов рисунков кнопок лежит))
