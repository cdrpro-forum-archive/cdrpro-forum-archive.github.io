GMS_Maker
Маленькая программа для создания пустых gms файлов или копирования gms макросов в папку GMS

Создание пустого gms файла

Выберите название программы
Выберите место назначения файла
Введите имя макроса
Нажмите кнопку "Create GMS file"

Копирование макроса

Выберите название программы
Выберите место назначения файла
Нажмите кнопку "Select" для выбора макроса или перетащите макрос на текстовое поле
Нажмите кнопку "Copy GMS file"
