Solowejka / 07.07.2012, 14:24

Ради всего святого.
Если, выбирая эти настройки в окне Options, в правом верхнем углу вы видите на картинке именно тот макет, который вам нужен, то жмите ОК.
Добавляйте нужное количество страниц (в брошюрах должно быть кратно четырём).
Теперь смотрите: если бы это был традиционный макет, то для всех нечётных страниц переплёт будет слева, для чётных — справа.
Для вашего изделия вполне можно использовать этот макет, просто готовое изделие будет повёрнуто на 90 град. по ч.с.
Верстайте страницы так, чтобы всё содержимое было повёрнуто к переплёту "головой" (верхом) для нечётных страниц и "ногами" (низом) — для чётных.
Таким образом, на всех страницах содержимое должно быть повёрнуто на 90 град. против ч.с.
Для удобства работы можно создать рабочий файл с нормальным расположением содержимого, то есть с ландшафтной (альбомной) ориентацией страниц.
А в печатном файле всё поверните как надо, заодно выполните и другие необходимые преобразования, предъявляемые к печатным макетам.
