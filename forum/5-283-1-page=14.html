И вот ещё: по дефолту цветовое пространство RGB, как заставить открывать новый документ в CMYK? Выставление галочек по адресам Options - Document - General - Default color mode - CMYK и Options - Document - Save options for the documents as defaults for new documents, ни к чему не приводит. В Default color management тоже естественно CMYK стоит праймари (XP SP3)

Там какой-то глюк в Corel, у меня вылечилось следующим образом.

В Workspace поставить галку Show New Document dialog box, при создании нового документа надо ОБЯЗАТЕЛЬНО пощелкать Primary color mode туда-обратно, без этого ставится цветовая модель RGB по умолчанию. В нее же и сохраняются настройки по умолчанию для новых документов. (Файл \путь к папке установки CorelDraw X5\\Draw\CORELDRWRGB.CDT).

Если этого не сделать, то, по-видимому, происходит следующее: при загрузке подтягивается модель CMYK по умолчанию, (файл \путь к папке установки CorelDraw X5\\Draw\CORELDRW.CDT), а сохраняется в модель RGB. В результате не удается сохранить настройки по умолчанию для новых документов.
