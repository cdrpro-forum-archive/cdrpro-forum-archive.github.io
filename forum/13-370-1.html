Макросы и скрипты
Много тем было про то как вставить макет из иллюстратора в корел для последующего редактирования. Проблема решается макросом Copy To Adobe Illustrator, который так же позволяет копировать и наоборот из корела в иллюстратор. Однако часто возникают проблемы со шрифтами. Они либо вообще не вставляются в иллюстратор, либо кракозябрами. Можно конечно переносить текст как кривые, но это не выход если потом потребуется редактирование текста. Немного поковырявшись в вышеупомянутом макросе, я нашёл способ заставить его переносить и текст с сохранением форматирования и возможностью последующего редактирования. В редких случаях требуется заменить пропавшие шрифты, потому что по каким то причинам некоторые шрифты имеют разные имена в кореле и иллюстраторе. К тому же заметил что переносятся маски, прозрачность, режимы наложения и т.д.

Итак, открываем макрос в редакторе VBA, и с помощью поиска, в модуле CopyPaste, находим строку With Activedocument.PDFSettings

и заменяем код, начиная с With Activedocument.PDFSettings до End With на следующий:

With Activedocument.PDFSettings
    .BitmapCompression = 3 ' CdrPDFVBA.pdfZIP
    .ColorMode = 3 ' CdrPDFVBA.pdfNative
    .EmbedFonts = True
    .EmbedBaseFonts = True
    .TrueTypeToType1 = True
    .SubsetFonts = True
    .SubsetPct = 80
    .CompressText = True
    .FileInformation = False
    .Encoding = 1 ' CdrPDFVBA.pdfBinary
    .Hyperlinks = False
    .IncludeBleed = False
    .Linearize = True
    .MaintainOPILinks = True
    .Overprints = True
    .pdfVersion = 8 ' CdrPDFVBA.pdfVersion17
    .PublishRange = pdfSelection
    .RegistrationMarks = False
    .SpotColors = True
    .Startup = 0 ' CdrPDFVBA.pdfPageOnly
    .SubsetFonts = False
    .TextAsCurves = False
    .TextExportMode = 0 ' CdrPDFVBA.pdfTextAsUnicode
    .Thumbnails = False
    .UseColorProfile = False
End With
В настройках макроса нужно снять все галочки кроме использования PDF формата.
Тестировалось на CD X4 и AI CS4
