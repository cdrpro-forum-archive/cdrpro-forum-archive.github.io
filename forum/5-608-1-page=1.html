Форматы записи CD, отражающие историю развития этой технологии, описаны в спецификациях, которые по традиции называются <цветными книгами>. Из них наиболее известны:
Yellow Book (желтая книга) - формат обычного CD-ROM (Compact Disc-Read Only Memory). Стандарт разработан Philips и Sony для использования CD в качестве носителя информации для ПК;
Red Book (красная книга) - стандарт разработан Philips и Sony для музыкальных компакт дисков CD-DA (Compact Disc-Digital Audio);
Green Book (зеленая книга) - формат CD-I (CD-Interactive), разработанный Philips для интерактивных мультимедийных приложений. Применяется для записи VideoCD в формате MPEG;
Orange Book (оранжевая книга) - стандарт CD-R , разработанный Philips и Sony для устройств CD-MO (Compact Disc-MagnetoOptical) и CD-WO (Compact Disc-Write Once).
Существует большое количество форматов записи CD. Рассмотрим наиболее распространенные:
CD-ROM - стандартный формат для записи информации на диск. Может иметь файловую систему ISO 9660 или Joliet. Формат записи секторов может быть CD-ROM Mode 1 или CD-ROM XA Mode 2;
CD-DA (Audio CD) - стандартный формат для записи музыки на диск. В этом формате на диск можно записать объем информации до 74 мин. Существует расширение этого стандарта CD-Text, позволяющее дописывать дополнительную текстовую информацию о произведениях и исполнителях;
Mixed-Mode CD - формат записи, при котором на первой дорожке записаны данные, а на остальных - музыка, причем запись осуществлена за одну сессию. Если такой диск проигрывать на обычной аудиоаппаратуре, первая дорожка будет звучать как резкий и неприятный шум. В силу этого чаще используется более удобный формат CD Extra;
CD Extra - формат записи, при котором в первой сессии записываются несколько музыкальных дорожек, а во второй сессии - дорожка с данными в формате CD-ROM XA. Такой тип записи часто используют производители компьютерных игр. Диски CD Extra можно проигрывать в обычной аудиоаппаратуре без отрицательных эффектов, свойственных Mixed-Mode CD;
CD-XA Bridge - формат записи, объединяющий CD-I и PhotoCD. Используется для записи Kodak PhotoCD и VideoCD.
Особняком стоит Multisession CD , который не является самостоятельным форматом, а представляет собой способ записывать информацию на диск не сразу, а порциями, которые называются сессиями. Запись в несколько сессий применяется в различных форматах. Для чтения таких дисков необходимы Multisession CD-ROM, каковыми являются все современные приводы за исключением самых старых 1- или 2-скоростных. Использование нескольких сессий связано с экономией дискового пространства: запись первой сессии отнимает 22 Мбайта, а каждой последующей - 13 Мбайт.
